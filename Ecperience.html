My Projects

CMU FIGHTER

CMU FIGHTER is a two-player fighting game inspired by Chiang Mai University.

Each character is uniquely crafted to reflect the distinct traits of different faculties within the university, complete with their own unique fighting moves.

Our team, including myself, developed the game using Scratch. You can check it out on GitHub


Tom-Khai

Tom-Khai is a text-based game in Thai that allows players to make choices that influence the storyline.

The decisions made throughout the game affect the progression of the narrative, leading to multiple possible endings.

Our team, including myself, developed the game using C++. You can check it out on GitHub
